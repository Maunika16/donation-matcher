# Donation Matcher (React + Vite)

Donation Matcher is a web application that helps connect donors with people or organizations in need by organizing and filtering donation items efficiently.

## Features
- Add donation items (food, clothes, books, etc.)
- View list of available donations
- Filter donations by category
- Clean and responsive UI
- Component-based React structure

## Tech Stack
- React
- Vite
- JavaScript (ES6)
- CSS

## Project Structure

src/
 components/
   AddDonation.jsx
   DonationList.jsx
   Filter.jsx
 data/
   donations.js
 styles/
   App.css

## How to Run Locally

1. Clone the repository
git clone https://github.com/Maunika16/donation-matcher.git

2. Open the project folder
cd donation-matcher

3. Install dependencies
npm install

4. Start development server
npm run dev
